## SEN0170 — Αισθητήρας Ταχύτητας Ανέμου

### Τεχνικά Χαρακτηριστικά
| Παράμετρος | Τιμή |
|---|---|
| Τύπος εξόδου | Αναλογική τάση |
| Εύρος εξόδου | 0 – 5 V |
| Εύρος ταχύτητας | 0 – 30 m/s |
| Ανάλυση | 0,1 m/s |
| Σφάλμα  | ±3% |

### Ανάλυση Datasheet
Ο αισθητήρας έχει **γραμμική** σχέση τάσης–ταχύτητας:

- Στα **0 V** → **0 m/s** (ακινησία)
- Στα **5 V** → **30 m/s** (μέγιστη ταχύτητα)

Η κλίση της γραμμής:

```
κλίση = 30 m/s ÷ 5 V = 6 (m/s)/V
```

### Τύποι
```
Τάση (V)      = analogRead × (5,0 ÷ 1023)
Ταχύτητα (m/s) = Τάση × 6
```

Συνδυασμένος τύπος:
```
Ταχύτητα (m/s) = analogRead × (5,0 ÷ 1023) × 6
```

### Σημείωση
Η γραμμική σχέση είναι εγγυημένη από τον κατασκευαστή για ολόκληρο το εύρος 0–5V.
Δεν απαιτείται βαθμονόμηση.
